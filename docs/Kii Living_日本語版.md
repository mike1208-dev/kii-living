# Kii Living

## 不動産会社向け コーポレートサイト（静的・高速）

> **概要**  
> 株式会社Kii Living のコーポレートサイトを、静的サイトジェネレーター Astro でハンドコード構築。不動産売買・仲介／リフォーム／不動産DX／住宅ローンの4事業を分かりやすく伝え、問い合わせにつなげる、軽量・高速・低コストなブランドサイトを実現します。テンプレート然としない和のコーポレート表現と、ローカル検索に強い構造化データを両立します。

---

# 課題

## ビジネス課題

- 4つの事業（売買・仲介／リフォーム／不動産DX／住宅ローン）を、ひと目で・信頼感を持って伝えたい
- 「相談してみよう」と思わせる、電話・メールへの明確な問い合わせ導線が必要
- チェーン店のようなテンプレート感を避け、ブランドとして読めるデザインにしたい
- 会社概要（所在地・代表者・宅建業免許番号など）を正確に掲載する必要がある

## 技術課題

- 第一印象を左右する「表示の速さ」と「モバイルでの崩れなさ」を最優先したい
- サーバー運用の手間やコストをかけず、堅牢に公開し続けたい
- 「不動産会社」としてローカル検索・地図・SNS共有で正しく見つかる必要がある

# ソリューション

## 担当

- Frontend Developer（設計・実装・デプロイ）
- ブランド / UI デザイン
- コンテンツ設定・運用引き継ぎ

## 実装内容

### ブランド・デザイン

- 明朝（見出し）× ゴシック（本文）の和のコーポレート表現に、深いエバーグリーンのアクセント（住まい・信頼）を採用
- OSのライト / ダークテーマに追従

### ページ構成

- 1ページ構成：ヒーロー → 事業内容（4カード）→ 会社概要 → お問い合わせ → フッター
- 各事業は「01〜04」の番号付きカードで、簡潔な説明とともに提示

### レスポンシブ

- CSS Grid / Flex で、モバイル・タブレット・PC のいずれでも崩れない1つのレイアウトを構築

### パフォーマンス

- Astro によりビルド時に HTML を生成（静的）。JS は必要な箇所にだけ（アイランド）投入し、初期表示を高速化
- 画像は最適化して配信

### SEO・構造化データ

- `RealEstateAgent` / `LocalBusiness` の JSON-LD 構造化データ、OGP、メタ情報を最適化し、ローカル検索とSNS共有に対応

### アクセシビリティ

- キーボード操作・フォーカス表示・コントラストに配慮

### 問い合わせ導線

- 電話番号・メール（または外部フォーム）への明確な導線を各所に配置

### デプロイ

- Cloudflare Pages への静的配信。Git push で自動ビルド・自動デプロイ（高速・低コスト・保守レス）

# 成果

- 4事業と会社の信頼性がひと目で伝わる、ブランドとして読めるコーポレートサイト
- 電話・メールへの明確な導線で、問い合わせにつながる構成
- 静的生成による高速表示とモバイル最適化で、第一印象を損なわない

# 技術的なポイント

### 静的だからこそ、速い・安い・堅牢

ビルド時に HTML を生成する静的構成により、表示が速く、ホスティングも安価で堅牢です。サーバー保守が不要で、コーポレートサイトに求められる「常に安定して速い」を無理なく満たします。

### 必要な場所にだけ JavaScript を（Astro アイランド）

Astro のアイランド構成で、JS は本当に必要な箇所にだけ読み込みます。軽量さを保ったまま、将来的な機能追加にも耐える構造です。

### ブランドとして読めるデザイン

明朝 × ゴシックの組み合わせと深いエバーグリーンのアクセントで、「一目でテンプレート」とわかる見た目を避け、和のコーポレートとしての信頼感を表現します。

### 見つけてもらう設計

`RealEstateAgent` / `LocalBusiness` の構造化データと OGP を実装し、ローカル検索や地図、SNS共有で「不動産会社」として正しく表示されるよう最適化します。

### 既知のトレードオフと発展

静的構成のため、コンテンツ更新には開発者の対応が必要です（お知らせや事業内容の頻繁な自己編集が必要になった場合は、ヘッドレスCMS の追加で対応可能）。まずは軽量・高速で立ち上げ、必要になった段階で拡張する方針です。
